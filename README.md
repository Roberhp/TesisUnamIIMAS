# Evaluación de modelos ligeros combinados para la detección de trastornos de salud mental en redes sociales

Detección de trastornos de salud mental usando modelos clásicos y una Feature Attention Network (FAN).

## Estructura
- src/: código principal
- scripts/: ejecución
- data/: datasets
- outputs/: resultados

## Ejecución
python scripts/train_model.py


## Requisitos adicionales (NLTK)
Después de instalar dependencias:
python scripts/setup_nltk.py
